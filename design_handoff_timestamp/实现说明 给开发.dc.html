开发实现说明 · macOS

TimeStamp — 按快捷键插入当前时间

把已定稿的设置面板原型，落地为真正可注册全局快捷键、向任意文本框输入时间的菜单栏小工具。提供两条路径：最快上手的 Hammerspoon，以及完整独立的原生 Swift App。

需要实现的三件事

1
注册全局快捷键
无论焦点在哪个 App，按下组合键（如 ⌘⇧T）都能触发。
2
按格式生成时间字符串
读取用户设置的格式串（yyyy-MM-dd HH:mm 等），格式化当前时间。
3
把字符串送进当前文本框
模拟键盘输入，或写入剪贴板后模拟 ⌘V 粘贴。需要「辅助功能」权限。
最快 · 约 10 分钟

路径 A · Hammerspoon 脚本

适合自用、想立刻见效。安装 Hammerspoon（brew install --cask hammerspoon），授予「辅助功能」权限，把下面内容粘进 ~/.hammerspoon/init.lua，菜单栏点 Reload Config 即可。
-- ~/.hammerspoon/init.lua -- 时间格式：用 strftime 占位符（%Y 年 %m 月 %d 日 %H 时 %M 分 %S 秒） local dateFormat = "%Y-%m-%d %H:%M" -- 注册全局快捷键 ⌘⇧T hs.hotkey.bind({"cmd", "shift"}, "T", function() local stamp = os.date(dateFormat) -- 方式一：逐字符模拟输入（兼容性最好） hs.eventtap.keyStrokes(stamp) -- 方式二（更快、适合长文本）：写剪贴板后模拟 ⌘V -- hs.pasteboard.setContents(stamp) -- hs.eventtap.keyStroke({"cmd"}, "V") end) hs.alert.show("TimeStamp 已就绪 ⌘⇧T")
改快捷键 → 改 bind 的修饰键和按键；改格式 → 改 dateFormat。strftime.org 有全部占位符对照。
可分发 · 独立 App

路径 B · 原生 Swift 菜单栏 App

适合做成能给别人用的成品，UI 即定稿的那个面板。技术选型：
  • 菜单栏 + 面板：MenuBarExtra（SwiftUI，macOS 13+），面板用 .menuBarExtraStyle(.window) 装定稿 UI。
  • 全局快捷键：用 KeyboardShortcuts 库，自带录制控件，正好对应面板里的「点击录制」交互。
  • 输入文本：CGEvent 模拟键盘，或写 NSPasteboard + 模拟 ⌘V。需请求 Accessibility 权限。
  • 格式化：DateFormatter，格式串与原型完全一致（yyyy/MM/dd/HH/mm/ss/EEE/a）。
① 定义快捷键 + 监听
import SwiftUI import KeyboardShortcuts extension KeyboardShortcuts.Name { static let insertTimestamp = Self("insertTimestamp", default: .init(.t, modifiers: [.command, .shift])) // ⌘⇧T } final class AppState: ObservableObject { @AppStorage("dateFormat") var dateFormat = "yyyy-MM-dd HH:mm" init() { KeyboardShortcuts.onKeyUp(for: .insertTimestamp) { [weak self] in self?.insertNow() } } func insertNow() { let fmt = DateFormatter() fmt.dateFormat = dateFormat TextTyper.type(fmt.string(from: Date())) } }
② 输入文本（剪贴板 + 模拟 ⌘V，最稳）
import AppKit enum TextTyper { static func type(_ text: String) { let pb = NSPasteboard.general let saved = pb.string(forType: .string) // 备份原剪贴板 pb.clearContents() pb.setString(text, forType: .string) let src = CGEventSource(stateID: .combinedSessionState) let vDown = CGEvent(keyboardEventSource: src, virtualKey: 0x09, keyDown: true)! // V let vUp = CGEvent(keyboardEventSource: src, virtualKey: 0x09, keyDown: false)! vDown.flags = .maskCommand vUp.flags = .maskCommand vDown.post(tap: .cgAnnotatedSessionEventTap) vUp.post(tap: .cgAnnotatedSessionEventTap) // 0.1s 后还原剪贴板 DispatchQueue.main.asyncAfter(deadline: .now() + 0.1) { pb.clearContents() if let saved { pb.setString(saved, forType: .string) } } } }
③ 菜单栏入口 + 定稿面板
@main struct TimeStampApp: App { @StateObject private var state = AppState() var body: some Scene { MenuBarExtra("TimeStamp", systemImage: "clock") { SettingsPanel() // ← 定稿的那个面板 UI .environmentObject(state) } .menuBarExtraStyle(.window) // 下拉成自定义面板 } }
⚠️
必须申请「辅助功能」权限。首次运行用 AXIsProcessTrustedWithOptions 弹窗引导用户到 系统设置 › 隐私与安全性 › 辅助功能 勾选本 App，否则模拟键盘无效。分发时需在 Info.plist 说明用途。
